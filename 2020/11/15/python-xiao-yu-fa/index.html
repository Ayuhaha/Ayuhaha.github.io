1.算术运算及赋值

>>> 9//2
4

>>> 9/2
4.5

>>> 8/2
4.0

>>> 3**2
9

>>> x,y,z=1,2,3
>>> x
1
>>> x*z
3

>>> .1+.1+.1
0.30000000000000004

>>> print(type(.3))
<class 'float'>

2.bool值

>>> print(.1+.1+.1==0.3)
False

>>> 1<2 and 1>3
False

>>> 1<3 or 3<5
True

>>> not 2>3
True

3.字符串

>>> "hahaha"
'hahaha'

>>> 'la\lala'
'la\\lala'

>>> 'I am learning python'
'I am learning python'

>>> 'I\'am learning python'       '\'让python跳过下个字符
"I'am learning python"        

>>> x='I am a boy'
>>> type(x)
<class 'str'>

4.类型和类型转换

>>> type(True)
<class 'bool'>

>>> type(int(False))
<class 'int'>

>>> type(float('123'))
<class 'float'>

5.字符串方法

>>> First_word='Hello'
>>> Secend_word='world'
>>> print(First_word+' '+Secend_word)
Hello world

>>> print(Secend_word*5)
worldworldworldworldworld

>>> print(len(Secend_word*5))
25

>>> Mom_said='Tom is a good boy.'
>>> Mom_said.islower()
False
>>> Mom_said.count('o')
4
>>> Mom_said.find('o')
1
>>> Mom_said.find('l')
-1

6.列表list

>>> A_list_of_things=['Hello',1,2,3,4.0,5.0,6.0,True,False]

>>> A_list_of_things[1]
1

>>> A_list_of_things[-1]
False

>>> len(A_list_of_things)
9

>>> A_list_of_things[len(A_list_of_things)-1]
False

>>> A_list_of_things[1:3]
[1, 2]

>>> A_list_of_things[:3]
['Hello', 1, 2]

>>> A_list_of_things[3:]
[3, 4.0, 5.0, 6.0, True, False]

7.in 和 not in

>>> 'This'in'This is my life!'
True

>>> 5 not in [1,2,3,4,5,6]
False

8.列表list和字符串string的可变性

>>> Hello='Hello hi~'
>>> Hello[1]
'e'

>>> list1=[1,2,3,4,5,6]
>>> list1[1]
2

>>> list1[0]='one'
>>> print(list1)
['one', 2, 3, 4, 5, 6]

>>> Hello[0]='M'
Traceback (most recent call last):
File "<pyshell#6>", line 1, in <module>
Hello[0]='M'
TypeError: 'str' object does not support item assignment

>>> Hello='Mello hi~'
>>> print(Hello)
Mello hi~

>>> AYU_said='Hello!My name is AYU.'
>>> haha=AYU_said
>>> haha
'Hello!My name is AYU.'
>>> AYU_said='Hello!My name is not AYU.'
>>> haha
'Hello!My name is AYU.'

>>> score=['A','B','C','D']
>>> score2=score
>>> score2
['A', 'B', 'C', 'D']
>>> score[0]='B'
>>> score
['B', 'B', 'C', 'D']
>>> score2
['B', 'B', 'C', 'D']

9.列表函数

>>> stock_price=[1,4,6,9,5,3,7,4,8,10,99]
>>> len(stock_price)
11
>>> max(stock_price)
99
>>> min(stock_price)
1

>>> sorted(stock_price)
[1, 3, 4, 4, 5, 6, 7, 8, 9, 10, 99]
>>> sorted(stock_price,reverse=True)
[99, 10, 9, 8, 7, 6, 5, 4, 4, 3, 1]

>>> card=['A','A','G','F','D','H']
>>> sorted(card)
['A', 'A', 'D', 'F', 'G', 'H']
>>> sorted(card , reverse=True)
['H', 'G', 'F', 'D', 'A', 'A']
>>> card
['A', 'A', 'G', 'F', 'D', 'H']

10.列表方法join和append


>>> a=['A','B','C','D','E']
>>> b='\n'.join(a)
>>> print(a)
['A', 'B', 'C', 'D', 'E']
>>> print(b)
A
B
C
D
E

>>> '\n'.join(a)
'A\nB\nC\nD\nE'
>>>'I\'am a student'
"I'am a student"

>>> c='-----'.join(a)
>>> print(c)
A-----B-----C-----D-----E

>>> name=['Tom','Jerry','AYU','Tom']
>>> name.append('Mike')
>>> print(name)
['Tom', 'Jerry', 'AYU', 'Tom', 'Mike']
>>> name.append('Mike')
>>> print(name)
['Tom', 'Jerry', 'AYU', 'Tom', 'Mike', 'Mike']
